# PRICING TABLE

## About the Application

This is  a single page application that displays the pricing model to its users. It exploits pseudo-selectors which helps with interactivity and dynamic styles.

## Features of the Application 

- Clean and modern UI
- Responsive layout for desktop and mobile
- Hover animations for pricing cards
- Highlighted "Popular" plan
- Custom CSS variable for easy theming
- Features availability indicators

## Technologies Used

- HTML5
- CSS3
